# 📚 论文智能爬取与知识图谱分析系统
一站式论文爬取、数据分析、关系挖掘、可视化展示工具  
简洁易用、零代码门槛，支持 arXiv/OpenReview 双数据源，自动生成动态知识图谱与研究领域思维导图。

---
## ✨ 功能介绍
- 🌐 双数据源支持：可从 **arXiv**（公开论文库）、**OpenReview**（会议论文库）爬取论文
- 🔍 自定义配置：可自由设置搜索关键词、爬取论文数量（10~200篇）
- 📊 结构化展示：清晰呈现论文标题、作者、研究方向等核心信息
- 👥 动态知识图谱：作者合作关系可视化，彩色区分研究团队，连线粗细体现合作紧密程度
- 🧠 图形化思维导图：按研究方向自动聚类，直观展示论文分布与课题关联
- 🖥️ 双启动方式：支持 Windows 双击启动、全系统终端启动
- 🌍 局域网共享：同一网络下，可分享链接让他人直接访问使用

---
## 📁 项目文件说明
项目共包含5个核心文件，分工明确、无需额外配置：
| 文件名               | 作用说明                                  |
| -------------------- | ----------------------------------------- |
| `paper_system.py`    | 主程序文件，包含爬虫、分析、可视化全部逻辑 |
| `requirements.txt`   | 依赖库清单，启动时自动安装所需环境        |
| `start.bat`          | Windows 专用一键启动脚本（小白首选）      |
| `.gitignore`         | Git 忽略文件，避免上传无用运行缓存        |
| `README.md`          | 项目说明文档，包含使用方法、免责声明等    |

## 🚀 快速启动方式
### 方法1：Windows 双击启动（推荐小白，零代码）
1. 下载项目并解压后，打开项目文件夹
2. 找到 `start.bat` 文件，**双击即可启动**
3. 等待终端自动安装依赖，安装完成后会自动打开浏览器，进入系统界面

### 方法2：终端启动（全系统通用，标准方式）
适用于 Windows、Mac、Linux 所有系统，步骤如下：
```bash
# 1. 打开终端，进入项目文件夹（替换为你的项目实际路径）
cd 你的项目路径/Paper-Analysis-System

# 2. 安装依赖库（第一次启动需执行，后续可省略）
pip install -r requirements.txt

# 3. 启动系统
streamlit run paper_system.py
